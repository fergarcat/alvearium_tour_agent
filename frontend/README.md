# Frontend для Ratoncito Pérez Travel Planner

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd frontend/frontend
npm install
```

### 2. Настройка переменных окружения
Создайте файл `.env` в папке `frontend/frontend/`:
```env
VITE_API_URL=http://localhost:8001/api/v1
```

### 3. Запуск фронтенда
```bash
npm run dev
```

Фронтенд будет доступен по адресу: http://localhost:5173

## 🔧 Интеграция с бэкендом

### Требования
- Бэкенд должен быть запущен на порту 8001
- API endpoints должны быть доступны по адресу `http://localhost:8001/api/v1`

### Основные функции

1. **Автоматическая инициализация чата**
   - При загрузке страницы автоматически подключается к бэкенду
   - Начинает новый разговор с Ratoncito Pérez
   - Показывает приветственное сообщение

2. **Интерактивный сбор данных**
   - Отображает статус сбора данных профиля семьи
   - Показывает текущий шаг сбора информации
   - Поддерживает пошаговый диалог

3. **Обработка ошибок**
   - Показывает сообщения об ошибках подключения
   - Предоставляет возможность повторной попытки
   - Graceful fallback при проблемах с API

4. **Состояния загрузки**
   - Индикаторы загрузки при отправке сообщений
   - Блокировка интерфейса во время обработки
   - Анимации для лучшего UX

## 📱 Компоненты

### Chatbot
- Главный компонент чата
- Управляет состоянием разговора
- Интегрируется с API бэкенда

### ChatMessages
- Отображает сообщения чата
- Поддерживает Markdown форматирование
- Показывает статус сбора данных

### ChatInput
- Поле ввода сообщений
- Поддержка Enter для отправки
- Автоматическое изменение размера

## 🔌 API Integration

Фронтенд использует следующие endpoints:

- `POST /api/v1/agents/chat` - отправка сообщений
- `POST /api/v1/agents/conversations/start` - начало нового разговора
- `GET /api/v1/agents/conversations/{id}/status` - статус разговора
- `GET /api/v1/health/` - проверка здоровья API

## 🎨 Стилизация

Используется Tailwind CSS для стилизации. Основные классы:
- Responsive design
- Темная/светлая тема
- Анимации и переходы
- Адаптивная верстка
